Getting Started

1

Install dependencies

bash
npm install ngx-streaming-player hls.js dashjs
2

Configure the player (app.config.ts)

Call providePlayer() in your application config to set global defaults, theme and translations. All feature functions are optional - omit any you don't need.

typescript
import { providePlayer, withTheme, withDefaults, withTranslations } from 'ngx-streaming-player';

export const appConfig: ApplicationConfig = {
  providers: [
    providePlayer(
      // Global theme applied to every <ngx-sp-player> in the app
      withTheme({ primaryColor: '#E76F51', borderRadius: '12px' }),

      // Defaults applied when the per-player [config] doesn't override them
      withDefaults({ autoplay: false, enablePiP: true, enableKeyboard: true }),

      // Localise all UI strings (any key you omit keeps the English default)
      withTranslations({
        play: 'Reproducir (k)',
        pause: 'Pausar (k)',
        settings: 'Configuración',
        quality: 'Calidad',
        speed: 'Velocidad',
        captionsTracks: 'Subtítulos',
        auto: 'Automático',
        normalSpeed: 'Normal',
        subtitlesOff: 'Desactivado',
        live: 'EN VIVO',
      }),
    ),
  ],
};
3

Import the component

typescript
import { StreamingPlayerComponent } from 'ngx-streaming-player';

@Component({
  standalone: true,
  imports: [StreamingPlayerComponent],
  // ...
})
export class MyComponent { }
4

Use it in your template

Three ways to configure - pick the one that fits your use case:

Config object (full control)
typescript
playerConfig: PlayerConfig = {
  src: 'https://example.com/stream.m3u8',
  autoplay: false,
  muted: false,
  volume: 1.0,
  enablePiP: true,
  enableKeyboard: true,
  playbackRates: [0.5, 1, 1.5, 2],
  controlsLayout: { autoHide: true, autoHideDelay: 3000 },
  theme: { primaryColor: '#E76F51' },
};
html
<ngx-sp-player [config]="playerConfig"></ngx-sp-player>
Individual inputs (simple)
html
<ngx-sp-player
  [src]="streamUrl"
  [autoplay]="true"
  [muted]="false"
  [volume]="0.8">
</ngx-sp-player>
Mixed - hot-swap via binding
html
<!-- Changing currentSrc triggers automatic hot-swap -->
<ngx-sp-player
  [config]="baseConfig"
  [src]="currentSrc">
</ngx-sp-player>
